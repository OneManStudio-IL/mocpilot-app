## 🚀 MOCPilot 2026.3.1 — Gamepad & Builder Update

We’re happy to announce the release of **MOCPilot 2026.3.1**, available on **Android**, **iOS**, and **macOS**.

This release brings major improvements to the block builder (touch + desktop) and stronger gamepad support.

---

<!-- truncate -->

## ✨ What’s New in 2026.3.1?

### 🎮 Gamepad Improvements

- Gamepad blocks now **auto-enable** when a profile is opened (including when a gamepad is already connected)
- Fixed an issue where **newly added gamepad event blocks** wouldn’t activate until restarting the program
- Added **gamepad value range** setting for finer control

### 🧱 Builder & Editor Improvements

- Big set of improvements to **touch gestures** and overall builder interactions (mobile + desktop)
- More reliable **undo/history** behavior (currently **dashboard only**), with a reworked history that tracks changes so you can return to a previous state more consistently
- Fixed multiple **port editor** and value editing issues on mobile
- Fixed an issue on mobile where pinch gestures could cause the **program area to get stuck**
- Improved block snapping and insertion previews when building control structures
- Added a **halo** for running blocks

### 🔌 Hub & Connectivity

- The **hub button selector** in the relevant block is now **always visible**, with reworked button event setup

### 🧩 Palette & Help

- Fixed palette issues after **locale/language switch** (could cause palette action buttons to become inactive)
- Added a blocks context menu option **“About Block”** to open documentation for the selected block

### 📚 Help Updates

- Help pages expanded with **direct links to blocks documentation**

### 🤖 Android Fixes

- Fixed issues with **sharing and importing profiles** on Android

### 🛠 Rendering & Performance

- Fixed **blurry/non-crisp blocks** at high scale by reworking renderer texture handling
- Optimized the **texture upscaler** for mobile devices and improved program area spacing/zoom behavior

---

Update to **2026.3.1** for a much smoother builder experience and better gamepad workflows.
