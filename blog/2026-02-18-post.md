## 🚀 MOCPilot 2026.2.2 — Expanded Hub & Control Update

We’re happy to announce the release of **MOCPilot 2026.2.2**, now available on **Android**, **iOS**, and **macOS**.

This release significantly expands hardware compatibility and introduces powerful new programming blocks for advanced hub interaction.

## ✨ What’s New in 2026.2.2?

### 🆕 New Hub Support

- **Mould King 4.0 hubs** are now fully supported  
- **Mould King 6.0 hubs** are now fully supported  
- **LEGO WeDo 2.0 hub** is now fully supported  

---

### 🔌 Sensor Support

- WeDo 2.0 Tilt Sensor  
- WeDo 2.0 Motion Sensor

---

### 🧱 New Programming Blocks

This release introduces new event and control blocks that expand real-time hub interaction capabilities.

#### 🔄 Hub Sensor Event Blocks
- **Hub Accelerometer Data Changed**  
- **Hub Tilt Changed**  
- **Hub Orientation Changed**  
- **All Hubs Battery Level Changed**

These blocks allow your program to react instantly to movement, rotation, and power changes.

#### 🔌 Hub Connection Control Blocks
- **Connect Hub**
- **Disconnect Hub**

You can now manage hub connections directly during program execution.  
This enables reconnecting a hub without restarting the program if the connection is lost, making projects more resilient and flexible.
