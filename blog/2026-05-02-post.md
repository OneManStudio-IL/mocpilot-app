## 🚀 MOCPilot 2026.3.2 — Dashboard Icons & Builder Fixes

We’re happy to announce the release of **MOCPilot 2026.3.2**, available on **Android**, **iOS**, and **macOS**.

This update improves dashboard controller visuals, fixes several builder interaction issues, and refines steering rack calibration behavior.

---

<!-- truncate -->

## ✨ What’s New in 2026.3.2?

### 🎛 Dashboard Improvements

- Added **new icons** for dashboard controllers

### 🧱 Builder & Block Fixes

- Fixed an issue where the **frequent hub tilt changed** event could misfire because of integer and double value comparison
- Fixed **block drag attach** issues affecting **C blocks** and other block types

### ⚙️ Steering Calibration Update

- Reworked the **calibrate steering rack** block
- The block now resets the **relative position to `0`** when calibration finishes
- Updated the calibration algorithm speed to **100**
