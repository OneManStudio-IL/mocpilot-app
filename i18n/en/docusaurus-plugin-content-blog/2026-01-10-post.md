## 🚀 MOCPilot 2026.1.0 — Feature & Stability Update

We’re happy to announce the release of **MOCPilot 2026.1.0**, now available on **Android**, **iOS**, and **macOS**.

This release introduces **guest mode**, improves usability on mobile devices, and delivers important stability and performance fixes across all supported platforms.

---

<!-- truncate -->

## ✨ What’s New in 2026.1.0?

### 🆕 New
- **Guest mode** — use the app without an account, with an option to create one later
- Language selection added to the authentication screen
- The app now blocks usage of hubs with unsupported firmware and shows a clear warning

### ⚙️ Improvements
- Improved block dragging behavior on mobile devices
- Blocks palette now hides automatically while dragging on mobile
- Alert dialogs now apply a blur effect to the background
- Improved text input handling across all platforms
- Smoother hub control with reduced latency

### 🛠 Fixes
- Fixed multiple block drag & drop issues, including stuck blocks and lost mouse/touch input
- Fixed crashes when stopping programs that use custom (“My”) blocks
- Fixed steering rack calibration not returning to center position
- Fixed hub control lag and delayed motor response
- Fixed issues with editor ports, sliders, and speed value blocks
- Fixed main dialog and various UI layout issues
- Updated translations and fixed minor visual inconsistencies

Update MOCPilot to the latest version and enjoy a smoother, more stable, and more accessible experience across all your devices.
