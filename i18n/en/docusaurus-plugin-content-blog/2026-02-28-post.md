## 🚀 MOCPilot 2026.2.3 — Dashboard & Hub Fixes

We’re happy to announce the release of **MOCPilot 2026.2.3**, available on **Android**, **iOS**, and **macOS**.

This update focuses on improving the dashboard workflow and resolving several important hub and editor issues.

---

<!-- truncate -->

## ✨ What’s New in 2026.2.3?

### 🧱 Dashboard Improvements

- You can now **duplicate dashboard controllers**, with options to **copy all properties** or **copy only by type**
- **Palette reworked** for faster building: shown underneath the dashboard, includes a clear button, stays open after adding components, and keeps section selection consistent

### 🔌 Hub Updates & Fixes

- **Mould King 4.0** reimplemented to the **new protocol on iOS** (Android continues using the previous protocol)
- Fixed an issue where the **“hub is connected” boolean block** didn’t appear while a hub was advertising
- Fixed a bug where the **light block** could stop working due to motor start/stop command handling

### 🛠 Fixes

- Fixed email validation in the input component that could allow invalid addresses like `mail@domain.`

---

Update MOCPilot to **2026.2.3** for a smoother dashboard experience and more reliable hub behavior across platforms.
